# Weather Diary (Дневник погоды)
## Описание программы
"Weather Diary" – это приложение с графическим интерфейсом, разработанное для ведения записей о погодных условиях. Пользователи могут регистрировать дату, температуру, описание погоды и информацию об осадках. Приложение позволяет просматривать, добавлять и фильтровать записи по дате или температуре. Все данные автоматически сохраняются в файле JSON, что обеспечивает их сохранность и удобство загрузки при следующем запуске.

## Инструкция по запуску

### Требования:

*   Python 3.x

*   Библиотеки: `tkinter`, `ttk`, `datetime`, `json`, `os`, `tkinter.messagebox`, `tkinter.filedialog` (входят в стандартную поставку Python)

### Запуск:

1.  Сохраните исходный код приложения (например, как `weather_diary_app.py`).

2.  Откройте терминал или командную строку.

3.  Перейдите в директорию, где вы сохранили файл.

4.  Запустите приложение командой:

    ```bash

    python weather_diary_app.py

    ```

5.  Приложение автоматически попытается загрузить данные из файла `weather_diary.json` (если он существует в той же директории). Сохраненные данные будут также использоваться для фильтрации и отображения.

## Примеры использования

*   **Добавление записи о погоде:**

    *   Введите дату в поле "Дата (YYYY-MM-DD)". По умолчанию установлено текущее число.

    *   Введите температуру в поле "Температура (°C)".

    *   Опишите погоду в поле "Описание погоды".

    *   Отметьте галочкой "Были осадки", если они были.

    *   Нажмите кнопку "Добавить запись". Новая запись появится в таблице ниже.

*   **Фильтрация записей:**

    *   Чтобы отфильтровать записи за определенную дату, введите её в поле "Фильтр по дате (YYYY-MM-DD)" и нажмите "Применить фильтр".

    *   Чтобы увидеть записи с температурой выше определенного значения, введите это значение в поле "Фильтр по температуре > (°C)" и нажмите "Применить фильтр".

    *   Для сброса всех фильтров и отображения всех записей нажмите кнопку "Сбросить фильтр".

*   **Сохранение и загрузка данных:**

    *   Нажмите кнопку "Сохранить в JSON" для ручного сохранения текущего состояния дневника в файл. Вам будет предложено выбрать место и имя файла.

    *   Нажмите кнопку "Загрузить из JSON" для загрузки предыдущих записей из файла. При запуске приложения данные загружаются автоматически из `weather_diary.json`.
